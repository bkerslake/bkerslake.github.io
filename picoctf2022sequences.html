picoCTF 2022 - Sequences.py Challenge Writeup




Challenge Description

“I wrote this linear recurrence function, can you figure out how to make it run fast enough and get the flag? Download the code here sequences.py Note that even an efficient solution might take several seconds to run. If your solution is taking several minutes, then you may need to reconsider your approach.”


Code Explanation and Analysis
Link to challenge code

We have an encrypted flag in a variable aptly named “ENCRYPTED_FLAG,” which we can only access by using value “Sol” to decrypt it. Unfortunately, Sol can only be found by finding the two millionth term of the recursive function “m_func,” and obviously that’ll take far too long to compute using the function’s current form. So, we must find a quicker way to access Sol without running the recursive function.


Solving:

So for starters, we’re actually also given a hint that helps us narrow down exactly how to approach this problem: “Google "matrix diagonalization". Can you figure out how to apply it to this function?”

After looking up matrix diagonalization, some of the examples mention the use of the Fibonacci sequence as an example of how recursive sequences can be defined using explicit formulae. And, when our recursive equation from the problem code is written out, it looks an awful lot like the Fibonacci sequence as well.
